# odin-recipes

This project guides me through creating a simple recipe website.
The main objective is to gain a solid understanding of HTML fundamentals
by practicing the use of various HTML elements such as headings, 
paragraphs, images, anchors, and more.

No part of the project features any CSS styling, since the segment 
covering it comes next in the curriculum. 

Images featured in the project were given appropriate credit, with links
back to the websites they were sourced from.
This usage falls under the category of non-commercial, educational use.
